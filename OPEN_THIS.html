🚀 Launch QUIZDOM

📋 How to Access QUIZDOM:

Option 1: Click the "Launch QUIZDOM" button above
Option 2: Open your browser and go to:
http://localhost:5000
Option 3: If you're on a remote server, try:
http://127.0.0.1:5000
Option 4: Copy and paste this URL in your browser:
http://localhost:5000

🔧 If it's not working:

Make sure the Flask application is running. You should see a terminal window with "QUIZDOM API is running" message.

If not, run this command in your terminal:

cd /workspace && source venv/bin/activate && python app.py
